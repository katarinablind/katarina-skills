Notes
Each vine is built fresh on every trigger (bloom-timeline pattern): old <g> removed, new paths appended with strokeDashoffset = length, then a single requestAnimationFrame later set to 0 — the CSS transition does the rest. Per-vine duration / delay / control-point jitter come from a deterministic seed (LCG keyed by vine index), so the same vine grows the same way every re-trigger. Geometry is a Catmull-Rom-style cubic Bezier chain with horizontal sway oscillating along the height; tip marker fades in at delay + dur · 0.85.
