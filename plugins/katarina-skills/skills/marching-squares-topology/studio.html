Field built once per frame, then iterated across all iso levels. Each cell maps a 4-bit corner index to 0–2 line segments via the canonical 16-case lookup. Endpoints are placed by linear interpolation along crossed edges, so contours read as smooth curves rather than blocky stairs. Hover the canvas to subtract a soft radial bowl from the field.
