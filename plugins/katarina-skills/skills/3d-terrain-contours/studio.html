Notes
One FBM height function, two meshes. The terrain pass writes vertex positions and per-vertex colors into a PlaneGeometry while caching heights into hCache. The contour pass runs marching squares on that same cache for each iso level, emitting LineSegments at iso × HSCALE + lift. Line buffer is preallocated at MAX_CSEGS = 24000; a cursor tracks fill, setDrawRange clips the rest.
