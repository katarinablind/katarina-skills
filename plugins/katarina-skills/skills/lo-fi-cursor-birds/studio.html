Notes
Each bird carries a stable baseVx, flapRate, and size. Cursor pull is acceleration with falloff ((d−inner)/range) and a speed cap; no lerp-to-target. When the cursor leaves, velocity exponentially blends back to baseVx over ~1s. An always-visible failsafe re-spawns the closest off-screen bird if all are out.
