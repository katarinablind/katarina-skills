Per-pixel raster fill. Each sample is bucketed via floor(t·N) against either a fixed or adaptive range, then written into an ImageData buffer that's blitted as one putImageData per frame. The outline overlay shares the same level thresholds, so each iso-line lands on a band boundary. Pair with marching-squares-topology for the line-only variant.
